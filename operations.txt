Operation History

Every important action should be recorded in an operation history file.

- Successful login.
- Failed login attempt.
- Machine started or stopped.
- Product produced.
- Product sold.
- Machine repaired.
- Employee added or removed.

Sample entries:

2026-07-15 10:30:25 | manager  | Started machine M-101
2026-07-15 10:45:10 | tech1    | Repaired machine M-101
2026-07-15 11:05:40 | sales1   | Sold product Shirt #25

Add one entry per line in the format:
YYYY-MM-DD HH:MM:SS | username | Description

Place this file in the project root so it can be used by administrators and for audit.
